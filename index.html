🔒 技術的な安全性の証明

本アプリは、ユーザーのプライバシー保護を最優先し、以下の**「クライアントサイド処理」**を採用しています。
  • ローカル実行: 顔認識ライブラリ face-api.js を使用し、すべての演算をブラウザ上で行います。
  • 画像データの非保持: FileReader API で取得した画像はメモリ内の一時的な処理に使用され、サーバーへのアップロードは一切行われません。
  • 静的サイト構成: 本サイトにはバックエンド（保存用DB）が存在しないため、物理的にデータを収集する仕組みがありません。
開発者ツールのネットワークタブを確認することで、通信が発生していないことを検証可能です。
